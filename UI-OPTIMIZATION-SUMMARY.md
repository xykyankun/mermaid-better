## 🎯 统一的设计语言

### 颜色方案
- **主要渐变**：蓝色(#2563eb) → 紫色(#9333ea) → 粉色(#ec4899)
- **背景**：slate-50 → blue-50 → purple-50
- **卡片**：白色 80% 透明度 + backdrop-blur
- **边框**：白色 20% 透明度

### 图标系统
- **库**：Lucide React
- **大小**：w-4 h-4 (小), w-5 h-5 (中), w-6 h-6 (大)
- **颜色**：跟随渐变主题

### 动画效果
- **背景光球**：animate-pulse（不同延迟）
- **按钮 hover**：scale-[1.02]
- **卡片 hover**：scale-[1.03]
- **加载状态**：animate-spin

### 圆角设计
- **按钮**：rounded-xl
- **卡片**：rounded-2xl
- **输入框**：rounded-xl
- **对话框**：rounded-2xl

### 阴影层次
- **基础**：shadow-lg
- **hover**：shadow-xl / shadow-2xl
- **强调**：shadow-2xl
